# 電気メガネ体感サイズ比較ツール データ (denkimeganetool-data)

このリポジトリは、Webツール「[電気メガネ体感サイズ比較ツール](https://megane.8796.jp/)」で使用されている機種プリセットデータ (`presets.js`) を管理するためのものです。

## 機種データの追加・修正について

新しいARグラス・スマートグラスのデータ追加や、既存データの修正に関するPull Request（プルリクエスト）を歓迎しています。
「この機種が載っていない」「スペックが間違っている」といった場合、ぜひご協力をお願いします。

### データの追加方法

`presets.js` を開き、`PRESETS` 配列の中に新しい機種データを以下の形式で追加してください。

```javascript
	{
		id: "device-id",              // 一意のID（英数字・ハイフン）
		label: "Device Name",         // 表示名
		resolution: { width: 1920, height: 1080 }, // 片目あたりの解像度
		diagonalFov: 46,              // 対角視野角 (度)
		color: "#3b82f6"              // テーマカラー (HEX)
	},
```

*   **アフィリエイトリンクは不要です**: リンク情報はツール側で別途管理しているため、このファイルには含めないでください。
*   **IDの重複に注意**: 既存のIDと被らないように設定してください。
